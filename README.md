# DineFlow

DineFlow is a Restaurant POS (Point of Sale) system designed to manage restaurant operations efficiently.

## Tech Stack

Frontend
- React
- Vite
- Redux
- Axios

Backend
- Node.js
- Express.js
- MongoDB

## Features

- Menu management
- Order management
- Table management
- Billing system
- Dashboard analytics

## Project Structure

frontend/ → React frontend application  
backend/ → Node.js Express backend APIs
